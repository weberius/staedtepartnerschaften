# ADR-008: Frontend als eigenständige Anwendung

## Status
Accepted

## Kontext
n8n ist kein geeignetes Werkzeug für interaktive Exploration oder Visualisierung.

## Entscheidung
Ein eigenständiges Frontend wird entwickelt:
- Angular
- Visualisierung von Graphen
- explorative Abfragen

n8n stellt keine Benutzeroberfläche bereit.

## Konsequenzen
- Höherer Entwicklungsaufwand
- Deutlich bessere UX
- Klare Trennung von Orchestrierung und Interaktion
