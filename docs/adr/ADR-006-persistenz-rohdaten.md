# ADR-006: Persistenz von Rohdaten

## Status
Accepted

## Kontext
Nachvollziehbarkeit erfordert Zugriff auf ursprüngliche Quellen.

## Entscheidung
Alle recherchierten Rohdaten werden persistiert:
- Web-Recherche
- strukturierte Quellen
- KI-generierte Texte

Die Rohdaten werden nicht überschrieben.

## Konsequenzen
- Reproduzierbarkeit von Analysen
- Höherer Speicherbedarf
- Möglichkeit zur erneuten Interpretation
