# ADR-010: Forschungsorientierter Fokus

## Status
Accepted

## Kontext
Das Projekt ist eine Studie, kein Produktivsystem.

## Entscheidung
Designentscheidungen priorisieren:
- Transparenz
- Nachvollziehbarkeit
- Reproduzierbarkeit

Nicht priorisiert:
- Performance
- Skalierung
- Hochverfügbarkeit

## Konsequenzen
- Gute Eignung für Forschung
- Begrenzte Produktreife
